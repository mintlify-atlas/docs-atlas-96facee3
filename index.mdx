## Welcome to Backend Template

Backend Template is a fully-configured Express.js server designed to accelerate your backend development. It comes with essential security middleware, database integration, and authentication utilities out of the box.

## Key Features

<CardGroup cols={2}>
  <Card
    title="Express.js v5"
    icon="bolt"
  >
    Modern Express.js server with security middleware including Helmet and CORS protection
  </Card>
  <Card
    title="PostgreSQL + Sequelize"
    icon="database"
  >
    Production-ready database integration with Sequelize ORM for relational data modeling
  </Card>
  <Card
    title="JWT Authentication"
    icon="shield"
  >
    Built-in JWT authentication with bcrypt password hashing for secure user management
  </Card>
  <Card
    title="Cloudinary Integration"
    icon="cloud"
  >
    Media management capabilities with Cloudinary for image and file uploads
  </Card>
  <Card
    title="Error Handling"
    icon="triangle-exclamation"
  >
    Centralized error handling for Sequelize validation, foreign key constraints, and custom errors
  </Card>
  <Card
    title="Dev & Prod Ready"
    icon="server"
  >
    Configured for both development (with nodemon) and production environments
  </Card>
</CardGroup>
